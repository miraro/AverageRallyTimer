本アプリは、ラリー競技における理想走行距離をリアルタイムに算出する高精度計算機です。実際の競技現場でのフィードバックを元に開発され、優勝実績を持つ実戦仕様のツールです。

主な機能：

CP/PC連動計算： スタート時刻や速度の修正に合わせて、予約されたPC通過時刻を自動で再計算。

WRC風高視認性UI： 激しい走行中でも瞬時に距離を読み取れる大型フォントを採用。

ナイトモード完備： 夜間ステージでも眩しくない配色切り替えが可能。

ログ保存： 過去の操作履歴を自動記録し、ステージ後の振り返りをサポート。

インストール不要。URLを開くだけで、あなたのスマートフォンやタブレットが最強のラリー機材に変わります。
